# Student Data Analysis Project

## Overview
A data analysis project built using Python, Pandas, and Matplotlib.
Performed data cleaning, analysis, and visualization on a 
self-created student dataset.

## Dataset
- Self-created dataset with 5 student records
- Columns: Name, Marks, City

## What I Did
- Data Cleaning — handled missing values (NaN)
- Removed duplicate entries
- Exploratory Data Analysis (EDA)
- Data Visualization using Matplotlib

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Learnings
- How to clean real-world messy data
- How to analyze and visualize data using Python
